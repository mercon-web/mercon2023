All accepted and presented papers will be submitted for publication to the IEEE Xplore. All papers should be written in English with a maximum paper length of 6 printed A4 pages. Use the following template while preparing full papers:

  • Micorosft Word (.doc) format
  • LaTex format - Zip, Tar
    • LaTeX (Bibliography Files - Zip, Tar
Use the following guidelines while submitting full papers:
  • All paper submissions are through EasyChair. Please submit papers using this link.
    • Please note that 1st time users need to create a new EasyChair account.
  • Manuscript submissions should be in Adobe Portable Document Format (PDF) only.
  • Manuscripts that do not conform to the above formatting guidelines will not be considered.
  • We request all authors to carefully revise the paper for language and formatting.
  • Please contact us if you have any questions about submitting your papers.
